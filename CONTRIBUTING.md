# Đóng góp

1. Tạo branch từ `main` và không commit `.pio`, `bin`, `obj`, token hoặc log chứa dữ liệu riêng.
2. Build firmware 4 MB và chạy các kiểm tra trong GitHub Actions.
3. Giữ tương thích giao thức USB protocol 2 và định dạng PCOTA1.
4. Ghi rõ module TFT/board đã kiểm tra thực tế trong pull request.
5. Không thay pin map, bảng phân vùng hoặc URL OTA mà không cập nhật tài liệu migration.
