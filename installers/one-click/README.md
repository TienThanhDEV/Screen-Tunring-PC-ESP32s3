# PCScreen one-click launchers

- macOS Intel: mở `INSTALL_PCSCREEN_MAC.command`.
- Windows x64: mở `INSTALL_PCSCREEN_WINDOWS.bat`.

Trình cài đặt hỏi trước khi xóa/nạp flash. Windows Agent tự chứa được tạo bởi GitHub Actions; nếu chỉ có mã nguồn, launcher sẽ mở builder và yêu cầu .NET 8 SDK. Vì việc chọn cổng USB và xác nhận xóa flash là thao tác an toàn bắt buộc, người dùng vẫn phải trả lời các câu hỏi trên màn hình.
